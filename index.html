🔒

파일이 같은 폴더에 있어도, 파일을 직접 더블클릭해서 열면

아래 방법 중 하나로 실행하면 정상 동작합니다.
✅ 방법 1 — VS Code Live Server (추천)
1. VS Code에서 폴더를 열기
2. 확장 프로그램 Live Server 설치
3. KN_Navigator_v1.html 파일에서
   우클릭 → Open with Live Server
✅ 방법 2 — Python 로컬 서버
파일이 있는 폴더에서 터미널(명령 프롬프트) 열기:

python -m http.server 8080

실행 후 브라우저에서
http://localhost:8080/KN_Navigator_v1.html
주소를 직접 입력하세요.
✅ 방법 3 — Chrome 보안 정책 우회 (임시)
Chrome 바로가기를 만들고 대상 경로 끝에 추가:

--allow-file-access-from-files

⚠ 보안상 취약하므로 업무용 PC에서만 임시 사용 권장.
해당 Chrome은 일반 인터넷 서핑에 사용하지 마세요.
필요한 파일: 2025대학별전형결과(규 수정보완본2).html (같은 폴더)
